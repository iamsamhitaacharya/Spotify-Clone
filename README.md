# Spotify Clone

A front-end clone of the Spotify web player built with plain HTML and CSS.



## What’s included

- Responsive sidebar, content area, and music player layout
- Spotify-inspired visual styling built with plain HTML and CSS
- Local assets for icons, album art, and cards
- Fixed music player and sticky navigation structure



## Preview

The current UI is deployed here: [https://iamsamhitaacharya.github.io/Spotify-Clone/](https://iamsamhitaacharya.github.io/Spotify-Clone/)



## Project Structure

- `index.html` - main page markup
- `style.css` - all styling for the layout and components
- `assets/` - images and icons used by the UI



## How to Run

1. Open `index.html` in your browser, or
2. Use VS Code Live Server if you want automatic refresh while editing



## Notes

- This is a static UI clone, so playback controls are visual only.
- External fonts and icons are loaded from CDNs.
